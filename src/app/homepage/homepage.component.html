Vítejte na webovém portálu Auctioner!

Důležité upozornění: Nejedná se o reálnou webovou službu! Tato webová aplikace byla navržena a implementována v rámci realizace diplomové práce na Mendelově univerzitě v Brně.

Základní informace

  • Jedná se o aplikaci aukčního portálu s možností přihazování v reálném čase
  • Systém dovoluje jednoduchou správu uživatelských účtů, stejně tak jako podporuje celý nákupní proces
  • Ohledně GDPR pojednává stránka Ochrana osobních údajů

Technické informace

  • Jedná se o moderní single page webovou službu skládající se z back-endové a front-endové části
  • Front-endová část je realizována za pomocí framerowku Angular, vzhledově se opírá o framework Bootstrap
  • Bank-endovou část věci poté zajišťuje Spring Bootová aplikace komunikující s angularem povětšinou skrze REST rozhraní. Určitým specifikem je přihazování v reálném čase, které se děje skrze websockety.
  • Uložení dat se děje skrze relační databázi PostgreSQL
